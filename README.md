# coc-project-Sudoku-Grid-Validator 
Project Description
This project validates all rows, coloumns and 3*3 sub - grids.
Accepts a hardcoded or user input 9*9 grid 
Clear output messages:  "Sudoku is valid"  "Sudoku is not Valid"
How to compile 
gcc main.c sudoku.c -o
sudoku_validator
How to run
./sudoku_validator
